# About

Malaysia's ATM location scraper, written in Elixir and Phoenix. [WIP].

# Installation

`$ mix ecto.create`

`$ mix ecto.migrate`

# Scrape Maybank ATM Locations

`$ mix scrape.maybank branch`

`$ mix scrape.maybank offsite`
